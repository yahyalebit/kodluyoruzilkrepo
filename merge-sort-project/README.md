# Merge Sort Projesi

## Proje 2

<h3>[16,21,11,8,12,22] -> Merge Sort</h3>

<h4>1. Yukarıdaki dizinin sort türüne göre aşamalarını yazınız.</h4>

```s
[16,21,11,8,12,22]
    /          \
[16,21,11]  [8,12,22]
    /   \       /  \
[16,21] [11] [8,12],[22]
  /  \    \    /   \    \ 
[16] [21] [11] [8] [12] [22]
  \  /     |    /   \   |
[16, 21] [11]  [8, 12] [22]
    \   /          \    /
[11, 16, 21]   [8, 12, 22]
       \            /
 [8, 11, 12, 16, 21, 22]
```
<h4>2. Big-O gösterimini yazınız.</h4>

```s
O(NlogN)
```
